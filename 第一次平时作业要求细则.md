使用神经网络增强图像重定向质量
### 题目描述
随着多种显示设备的普及（如手机、平板、电脑、电视），图像需要适应不同的宽高比。传统的图像缩放方法可能导致内容变形或重要信息丢失。Seam Carving 是一种内容感知的图像缩放技术，但原始算法对图像语义内容的理解有限。本实验旨在探索如何利用轻量级深度学习模型增强 Seam Carving 算法，在有限计算资源下实现更智能的图像重定向。要求结合卷积神经网络（CNN）改进显著性图的效果，应用 Seam Carving 算法进行图像重定向，并通过定量指标（如 PSNR 和 SSIM）评估重定向效果。
实验要求

### 显著性图生成与增强

掌握常规显著性检测方法及 CNN 在显著性图生成中的作用，探索结合多种显著性图（如 CNN 特征图、CAM 激活图、传统显著性图、边缘检测图）增强特征提取效果。
需分析不同显著性图的特点，并说明融合策略对重定向效果的贡献。


### Seam Carving 算法实现

实现基础 Seam Carving 算法，基于显著性图进行图像重定向。
缩放比例需控制在 40%-60% 范围内（即图像宽度或高度缩放至原始尺寸的 40%-60%），避免过小或过大缩放导致内容失真。
能量函数需结合图像梯度与显著性图，推荐公式为：能量 = 梯度 + 显著性。


### 数据集要求

用于测试的基准图片不少于 50 张，可从网络公开数据集（如 MSRA、PASCAL VOC）或自行收集获取。
图片内容需多样化，涵盖自然风景、人物、建筑等多种场景，确保实验结果的普适性。
每张图片需提供原始尺寸和重定向后的尺寸信息。


### 质量评估

将重定向后的图像重新缩放到原始尺寸（可使用插值算法，如双线性插值或最近邻插值），或用空白填充至原始尺寸。
计算以下两种方法的评估指标：

传统插值算法（如双线性插值或双三次插值）。
基于显著性图的 Seam Carving 算法。


每张图像需计算 PSNR（峰值信噪比）和 SSIM（结构相似性）得分，并以表格形式列出比较结果。
提交最终的平均 PSNR 和 SSIM 得分，并分析两者的差异及原因。

### 补充说明与注意事项

### 显著性图实现

可使用预训练的轻量级 CNN 模型（如 MobileNet、EfficientNet）提取特征图，降低计算复杂度。
CAM 激活图需明确说明生成方法（如 Grad-CAM 或 Score-CAM）。
传统显著性图可参考经典算法（如 Itti 模型、谱残差法）。
边缘检测推荐使用 Sobel 或 Canny 算子，确保边缘信息清晰。


### Seam Carving 实现

需实现动态规划或贪婪算法来寻找最小能量缝，确保算法效率。
可选优化：支持水平和垂直方向的 Seam Carving，需在报告中说明实现方式。


### 实验环境（推荐）

推荐使用 Python 编程语言，常用库包括 OpenCV、PyTorch/TensorFlow、scikit-image、numpy 等。
需在代码中说明使用的库版本及运行环境（如 Python 3.8、PyTorch 2.0 等）。
若使用 GPU 加速，需说明硬件配置。


### 评估指标计算

PSNR 和 SSIM 的计算可使用现成库（如 scikit-image 的 compare_psnr 和 compare_ssim），但需理解其原理并在报告中说明。
建议对结果进行可视化分析（如绘制 PSNR/SSIM 的分布图或直方图）。


### 常见问题与建议

若显著性图质量较差，可能导致 Seam Carving 删除重要区域，建议调整融合比例或优化 CNN 模型。
测试图片分辨率不宜过高（如控制在 512×512 以内），以降低计算量。
注意代码运行效率，避免因数据集过大或算法复杂导致运行时间过长。
